## Back up 
```shell
Move-Item $env:LOCALAPPDATA\nvim $env:LOCALAPPDATA\nvim.lazy
Move-Item $env:LOCALAPPDATA\nvim-data $env:LOCALAPPDATA\nvim-data.lazy
```

```shell
git clone https://github.com/postcypunk/LazyVim $env:LOCALAPPDATA\nvim
```
